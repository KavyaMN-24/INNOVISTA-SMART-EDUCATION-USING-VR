# INNOVISTA-SMART-EDUCATION-USING-VR
An immersive virtual reality-based education platform developed using Unity 3D and C# to enhance learning through interactive simulations.

This project focuses on building an interactive learning platform using **Virtual Reality (VR)** technology to enhance student engagement and understanding through immersive educational environments.

## Overview
Virtual Reality (VR) creates a simulated 3D environment that allows users to interact and explore concepts in an intuitive and experiential way. This project aims to make education more interactive and accessible, especially for students with disabilities, by providing realistic learning simulations.

## Features
- Developed using **Unity 3D** 
- Supports **VR Headset integration**
- Provides interactive modules for learning different subjects
- Includes accessibility features for inclusive learning

Programming Language: Java 
Tools: Visual studio code, Netbeans, Navicat 
Toolbox: Unreal engine, Trinus VR 
VR box / VR oculus 
Smart Phone with gyroscope
